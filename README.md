## Development

To use pipenv in vscode

```bash
export PIPENV_VENV_IN_PROJECT=1
```

```bash
pipenv install
```

to update the dependencies

```bash
pipenv update
```

to get the shell

```bash
pipenv shell
```

to isort

```bash
make isort
make isort-apply
```
